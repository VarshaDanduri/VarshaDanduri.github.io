CPU Rasterizer

C++ · Software Rendering · Computer Graphics

A software rasterizer built from scratch on the CPU. Starting from a hello triangle, the renderer grew to support 3D geometry, depth buffering, and shaded meshes: all without relying on a GPU graphics pipeline.
