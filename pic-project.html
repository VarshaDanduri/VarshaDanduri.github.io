Image Convolution

C · pthreads · Image Processing

A multithreaded C program that applies 3×3 convolution kernels to images. Given an input file and a filter type, it loads the image with stb_image, convolves each pixel against a kernel matrix, and writes the result as PNG. Six filters are supported (edge, sharpen, blur, Gaussian blur, emboss, and identity).

Each output pixel is a weighted sum of its 3×3 neighborhood. Edge pixels clamp to the nearest border value so the kernel never reads out of bounds. The edge-detection kernel highlights intensity changes, which is why high-contrast scenes like this produce strong line work.

Convolution runs in parallel using pthreads, splitting rows across worker threads. The same getPixelValue function drives every filter; only the kernel matrix changes between modes. 

Matrix algorithms[] = {
  {{0,-1,0},{-1,4,-1},{0,-1,0}},           // edge
  {{0,-1,0},{-1,5,-1},{0,-1,0}},           // sharpen
  {{1/9.0,1/9.0,1/9.0}, ...},              // blur
  {{1.0/16,1.0/8,1.0/16}, ...},            // gauss
  {{-2,-1,0},{-1,1,1},{0,1,2}},            // emboss
  {{0,0,0},{0,1,0},{0,0,0}}                // identity
};
